# Language Polishing

Use this reference when the user asks for润色、改写、压缩、规范化、去 AI 味、提高公文感, or when a draft has unclear, exaggerated, informal, or empty language.

## Polishing Modes

Choose the lightest mode that solves the problem:

- Normalize: fix punctuation, hierarchy, titles, terms, dates, and obvious wording errors.
- Tighten: remove repetition, shorten long openings, make tasks and responsibilities clearer.
- Formalize: replace colloquial or internet language with formal official wording.
- De-risk: downgrade unsupported conclusions, mark uncertain facts, remove invented authority or overpromising.
- Rebuild: restructure paragraphs when the draft's文种, logic, or section roles are wrong.

Do not beautify by adding new facts, stronger conclusions, or ornamental paired headings.

## Style Standard

Good公文 language is:

- Accurate: names, roles, numbers, time, and policy terms are precise.
- Concise: no repeated background, no empty transition paragraphs.
- Formal: institutional voice, not personal emotion or internet tone.
- Measured: judgments match evidence.
- Actionable: who does what by when is clear when the document requires action.

## High-Risk Language

Flag or soften:

- Unsupported praise: `显著成效`, `重大突破`, `全面完成`, `圆满完成`, `走在前列`.
- Vague action: `高度重视`, `切实加强`, `扎实推进`, `形成合力`, `持续发力`, `不断提升`.
- Unclear timing: `尽快`, `及时`, `适时`, `近期`, `原则上`.
- Over-authority: `必须无条件`, `一律`, `严禁`, `责令`, when the issuing body lacks that authority.
- Informal wording: internet slang, emotional adjectives, casual verbs, rhetorical questions.

These expressions are not always forbidden. Keep them only when facts, authority, and context support them.

## Rewrite Pattern

When polishing, preserve the user's facts and intent:

1. State the main risk or improvement direction.
2. Produce the revised draft or targeted paragraph.
3. List facts that still need confirmation.
4. Explain only the highest-impact edits, not every wording change.

## Examples Of Direction

- "高度重视" -> specify the responsible meeting, document, or action if available.
- "取得显著成效" -> name measurable outcomes or write "取得一定进展".
- "尽快完成" -> ask for or insert a concrete deadline placeholder.
- "必须无条件执行" -> use "请按要求抓好落实" unless coercive authority is confirmed.
