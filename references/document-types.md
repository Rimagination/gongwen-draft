# Document Types And Templates

Use this reference for common structures. Replace bracketed placeholders and remove unused sections.

## Routing Quick Table

| Purpose | Prefer |
| --- | --- |
| Request instruction or approval from a higher authority | 请示 |
| Report work, reflect situations, or answer inquiries | 报告 |
| Reply to a lower authority's request | 批复 |
| Deploy, issue, transmit, or require implementation | 通知 |
| Praise, criticize, or convey important situations | 通报 |
| Consult, inquire, request support, or reply between non-subordinate bodies | 函 |
| Record main meeting situation and agreed matters | 纪要 |
| Announce important/statutory matters domestically or abroad | 公告 |
| Publish matters to be observed or known within a scope | 通告 |
| Propose views and handling methods for important issues | 意见 |
| Make important decisions, deployments, rewards, punishments, or adjustments | 决定 |
| Meeting-approved important decision | 决议 |
| Publish regulations or major compulsory measures by authorized subject | 命令/令 |
| Publish important decisions, major matters, meeting outcomes, or statistics | 公报 |
| Submit matters to a people's congress procedure | 议案 |

Common formal materials:

| Purpose | Prefer |
| --- | --- |
| Summarize completed work, outcomes, experience, problems, and next steps | 工作总结 |
| Arrange a task with goals, division of labor, schedule, and safeguards | 工作方案 |
| Present field research findings and evidence-based suggestions | 调研报告 |
| Prepare meeting, inspection, or superior-level oral/written briefing materials | 汇报材料 |
| Report selected information in a concise internal bulletin style | 简报 |
| Submit time-sensitive or focused situation updates for leadership attention | 情况专报 |
| Prepare a role-specific speech or meeting address | 讲话稿 |
| Reply to a specific inquiry, suggestion, proposal, or request by letter | 回复函 |

Always verify行文方向 and authority. When user facts are insufficient, use placeholders rather than dropping required structures.

## 通知

Use for issuing, arranging, conveying, forwarding, or requiring implementation.

Structure:

```text
关于[事项]的通知

[主送机关/单位]：

为[目的]，根据[依据]，现就[事项]通知如下。

一、[总体要求/工作目标]
[说明背景、原则、目标。]

二、[主要任务/具体安排]
（一）[任务一]。[责任主体、完成时限、工作标准。]
（二）[任务二]。[责任主体、完成时限、工作标准。]

三、[工作要求]
（一）提高认识。
（二）压实责任。
（三）按时报送/抓好落实。

[发文机关]
[成文日期]
```

## 请示

Use to request instruction or approval from a higher authority. Keep one main matter.

Structure:

```text
关于[事项]的请示

[上级机关]：

因[背景和必要性]，根据[依据]，我单位拟[拟办事项]。现将有关情况请示如下。

一、基本情况
[事实、现状、问题，避免夸张。]

二、拟办意见
[具体方案、资金/人员/权限需求、风险控制。]

三、请示事项
恳请[上级机关]同意[具体请求]。

妥否，请批示。

[发文机关]
[成文日期]
```

Avoid: multiple unrelated matters, vague requests, reporting-only content, missing authority basis.

## 报告

Use to report work, reflect situations, or answer inquiries. Do not request approval.

Structure:

```text
关于[事项]的报告

[上级机关]：

根据[要求/背景]，现将[事项]有关情况报告如下。

一、基本情况
[事实、进展、数据。]

二、主要做法
[措施和过程。]

三、存在问题
[客观说明短板和原因。]

四、下一步工作
[具体安排。]

特此报告。

[发文机关]
[成文日期]
```

## 函

Use between non-subordinate organs for consultation, inquiry, reply, or requesting approval.

Structure:

```text
关于[事项]的函

[对方机关/单位]：

为[目的]，现就[事项]函告/商请如下。

一、[事项说明]
[背景、依据、需要对方知悉或协助的内容。]

二、[商请事项/回复意见]
[具体、清楚、可操作。]

请予支持/请研究函复。

[发文机关]
[成文日期]
```

## 纪要

Use to record meeting situation and agreed matters.

Structure:

```text
[会议名称]纪要

[时间]，[主持人]主持召开[会议名称]，研究[议题]。会议听取了[汇报]，讨论了[事项]，议定如下。

一、会议认为
[共识、判断。]

二、会议议定
（一）[议定事项一]。
（二）[议定事项二]。

三、会议要求
[责任单位、时限、落实机制。]

出席：[名单/单位]
列席：[名单/单位]
```

## 通报

Use for praise, criticism, conveying important spirit, or informing important situations.

Structure:

```text
关于[事项]的通报

[主送机关/单位]：

[事实经过/总体情况]。

一、基本情况
[客观事实。]

二、原因分析/经验启示
[根据通报目的选择。]

三、处理决定/工作要求
[表扬、批评、整改、学习推广等。]

[发文机关]
[成文日期]
```

## 意见

Use to present views and handling methods for important issues.

Structure:

```text
关于[事项]的意见

[主送机关/单位]：

为[目的]，根据[依据]，现就[事项]提出如下意见。

一、总体要求
[指导思想、目标、原则。]

二、主要任务
（一）[任务一]。
（二）[任务二]。

三、组织实施
[责任分工、保障措施、督促落实。]

[发文机关]
[成文日期]
```

## 决定

Use for important deployments, rewards/punishments, processing decisions, changes, or revocations.

Structure:

```text
关于[事项]的决定

[主送机关/单位]：

[作出决定的背景、依据和必要性]。现决定如下：

一、[决定事项]
二、[执行要求]

[发文机关]
[成文日期]
```

## 决议

Use for important decisions discussed and adopted by a meeting.

Structure:

```text
[会议名称]关于[事项]的决议

（[通过时间和会议名称]通过）

会议听取和审议了[事项]，认为[基本判断]。会议决定：

一、[决议事项一]
二、[决议事项二]
三、[贯彻要求]
```

## 命令（令）

Use only when the issuing subject has authority to publish regulations, announce major compulsory measures, appointments, or awards.

Structure:

```text
[发文机关]令

第[序号]号

[命令正文：公布事项、依据、生效时间或执行要求。]

[签署人/发文机关]
[成文日期]
```

If the user's issuing body lacks authority, explain the mismatch and suggest 通知、决定, or another suitable文种.

## 公报

Use to publicly release important decisions, major matters, meeting outcomes, or statistical information.

Structure:

```text
[事项]公报

[发布背景或统计/会议范围]。

一、[主要内容一]
二、[主要内容二]
三、[说明或结语]

[发布机关]
[发布日期]
```

## 公告

Use to announce important or statutory matters domestically or abroad.

Structure:

```text
关于[事项]的公告

[公告事项、法定依据、适用范围、生效时间。]

特此公告。

[发文机关]
[成文日期]
```

## 通告

Use to publish matters that should be observed or known within a certain scope.

Structure:

```text
关于[事项]的通告

为[目的/依据]，现就[事项]通告如下：

一、[适用范围]
二、[具体事项]
三、[执行要求/法律责任/有效期限]

特此通告。

[发文机关]
[成文日期]
```

## 议案

Use when a legally authorized subject submits matters to the people's congress or its standing committee for deliberation.

Structure:

```text
关于提请审议[事项]的议案

[审议机关]：

根据[依据]，现提请审议[事项]。

一、必要性
二、主要内容
三、提请审议事项

[提出机关]
[成文日期]
```

If the context is not a people's congress procedure, do not use 议案.

## 批复

Use to reply to a lower authority's请示.

Structure:

```text
关于[请示事项]的批复

[请示机关]：

你单位《关于[请示标题]》（[文号]）收悉。经研究，现批复如下：

一、[同意/原则同意/不同意/具体意见]
二、[办理要求或注意事项]

此复。

[发文机关]
[成文日期]
```

## 工作总结

Common structure:

```text
[单位/领域][时间]工作总结

一、主要工作和成效
[分条列明任务完成情况、数据、典型做法和实际成效。]

二、经验做法
[概括可复用机制，不夸大、不拔高。]

三、存在问题
[客观列出短板、原因和约束条件。]

四、下一步打算
[写明目标、措施、责任和时限。]
```

Use facts and representative examples. Do not overstate outcomes.

## 工作方案

Common structure:

```text
[工作名称]工作方案

为[目的]，根据[依据/部署]，结合[实际情况]，制定本方案。

一、总体要求
[指导原则、工作目标和适用范围。]

二、重点任务
（一）[任务一]。[责任单位、完成时限、工作标准。]
（二）[任务二]。[责任单位、完成时限、工作标准。]

三、实施步骤
（一）动员部署阶段。[时间、动作。]
（二）组织实施阶段。[时间、动作。]
（三）总结提升阶段。[时间、动作。]

四、保障措施
[组织领导、协同机制、督促检查、信息报送、风险防控。]
```

方案要可执行。Avoid only slogans; include responsible units, timeline, deliverables, and review mechanism where known.

## 调研报告

Common structure:

```text
关于[主题]的调研报告

一、调研背景和基本情况
[调研对象、范围、方式、时间和基本事实。]

二、主要发现
[按证据列出情况和特点。]

三、问题及原因
[区分事实问题、原因分析和待核实判断。]

四、对策建议
[提出可执行建议，尽量对应责任主体、步骤和条件。]
```

Recommendations should be feasible, assigned where possible, and grounded in evidence.

## 汇报材料

Common structure:

```text
关于[事项]的汇报材料

[汇报对象/会议名称]：

根据[要求/安排]，现将[事项]有关情况汇报如下。

一、基本情况
[背景、进展、基础数据。]

二、主要工作
[已经开展的措施、机制、成效。]

三、存在问题
[困难、短板、风险，不回避、不夸张。]

四、下一步安排
[后续措施、时限、责任、需要协调事项。]
```

Use a concise, upward-facing tone. Do not turn a汇报材料 into a请示 unless the task explicitly asks for approval.

## 简报

Common structure:

```text
[简报名称]
第[期数]期

[编发单位]                         [日期]

[标题]

[导语：一句话概括最重要事实。]

一、[动态/做法/成效一]
[短段落，写事实、动作、结果。]

二、[动态/做法/成效二]
[短段落，保持信息密度。]

三、[下一步/提示/要求]
[如无必要可删除。]
```

简报应短、准、实。Avoid long background explanations and unsupported评价.

## 情况专报

Common structure:

```text
关于[事项]的情况专报

[报送对象]：

[导语：说明事项、时间、地点、对象和当前状态。]

一、基本情况
[已核实事实、数据、进展。]

二、风险研判/原因分析
[区分事实、研判和待核实信息。]

三、工作建议/处置措施
[提出可操作措施、责任主体、时限和需要协调事项。]
```

Use for focused, time-sensitive reporting. Mark uncertain information as待核实 and avoid turning analysis into confirmed facts.

## 讲话稿

Common structure:

```text
[场合]讲话稿

一、提高认识，准确把握[主题]的重要意义
[结合场合和听众说明为什么要讲。]

二、突出重点，扎实推进[任务]
[部署重点任务、责任和方法。]

三、压实责任，确保各项工作落到实处
[强调组织领导、执行、监督和反馈。]
```

Use the speaker's role and audience. Avoid making commitments beyond authority.

## 回复函

Common structure:

```text
关于[事项]的回复函

[来函单位/个人]：

你方《关于[来函标题/事项]》收悉。经研究，现回复如下：

一、[办理情况/核实情况]
[说明已核实事实、办理依据和处理过程。]

二、[回复意见]
[逐项回应对方诉求或问题，写清是否采纳、如何处理、后续安排。]

三、[联系方式/后续沟通]
[如需要，写明联系人、沟通渠道或补充材料要求。]

[发文机关]
[成文日期]
```

回复函 should answer the exact incoming matter. Do not use批复 language unless replying to a subordinate机关的请示.
